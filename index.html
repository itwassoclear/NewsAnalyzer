Об авторе

Привет! Меня зовут Маша и я начинающий фронтенд-разработчик с бэкграундом дизайнера. Прошла учебный курс в Яндекс.Практикуме, а данный сайт - наглядный результат того, чему я успела научиться.

Во время обучения были выполнены проекты с использованием HTML, CSS, BEM, JS (ES5 / ES6), API, GIT, Webpack. Проекты были одобрены командой ревью Яндекс.Практикума. Каждый из них можно постреть здесь. Также, во время обучения совместно с командой студентов был реализован внеучебный проект по переносу сайта raklechitsa.ru с конструктора Tilda. Для реализации за короткое время с нуля были освоены следующие инструменты: Vue.js, Vuex, Nuxt.js и Git. Ссылка на командный репозиторий.
